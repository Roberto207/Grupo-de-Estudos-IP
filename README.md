# 📦 Grupo de Estudos de Introdução à Programação ao Machine Learning — BIA T7 UFG (2026)

Este repositório reúne todos os materiais oficiais de aula do **Grupo de Estudos de Introdução à Programação ao Machine Learning** do BIA T7/UFG, incluindo conteúdos preparados por monitores e apresentadores, apresentações, códigos, leituras recomendadas e atividades práticas.

Grupo organizado inicialmente pelos alunos do BIA T7 — Roberto, Beatriz e Carlúcio.

## 🤝 Como Contribuir

1. Faça um fork do repositório
2. Crie uma branch com o nome do encontro: `git checkout -b encontro-X-descricao`
3. Adicione seus materiais na pasta correspondente
4. Abra um Pull Request descrevendo o que foi adicionado
